# PidCat - A Python Process Monitoring Tool

PidCat is a Python process monitoring tool that displays a list of all running processes and their corresponding details such as process name, process id, parent process id, process status, memory usage, executable path, command line arguments, and creation time.

## Features

PidCat supports the following features:
- Process list: Displays a list of all running processes and their corresponding details.
- Filtering: Supports filtering processes by name, id, status, and memory usage.
- Sorting: Supports sorting processes by name, id, status, memory usage, and creation time.
- Refreshing: Supports automatic refreshing of the process list.
- Copying information: Supports copying of process information to the clipboard.
- Command line arguments: Displays the command line arguments of a process.
- Executable path: Displays the executable path of a process.
- Creation time: Displays the creation time of a process.
- Memory usage: Displays the memory usage of a process.

## Requirements

PidCat requires the following packages to be installed:
- Python 3.6 or later
- PyInstaller 6.16 or later
